# Developer Portfolio

## Project Overview

Personal developer portfolio showcasing web development projects and skills.

## Available Skills

This project has access to the following auto-installed design and animation skills:

### 3D & Graphics
- **threejs-webgl** - Three.js WebGL 3D development
- **react-three-fiber** - React Three Fiber 3D in React
- **babylonjs-engine** - Babylon.js game engine
- **aframe-webxr** - WebXR VR/AR experiences
- **lightweight-3d-effects** - Zdog, Vanta.js, Vanilla-Tilt.js
- **pixijs-2d** - Fast 2D WebGL rendering

### Animation & Motion
- **gsap-scrolltrigger** - GSAP scroll animations
- **motion-framer** - Framer Motion animations
- **react-spring-physics** - React Spring physics
- **animejs** - Anime.js animations
- **lottie-animations** - Lottie After Effects
- **animated-component-libraries** - Magic UI, React Bits

### Scroll & Transitions
- **locomotive-scroll** - Smooth scrolling
- **barba-js** - Page transitions
- **scroll-reveal-libraries** - AOS scroll reveals

### Design & Patterns
- **modern-web-design** - 2024-2025 design trends
- **web3d-integration-patterns** - Multi-library architecture

## Default Approach

For this portfolio, prefer:
- **gsap-scrolltrigger** for scroll-driven animations
- **motion-framer** for React component animations
- **lottie-animations** for decorative animations
- **scroll-reveal-libraries** for simple entrance animations
- **modern-web-design** for current design trends

Avoid heavy 3D unless specifically requested - focus on performant, elegant animations.
